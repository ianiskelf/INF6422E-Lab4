# Semantic and Context-Aware Security Analysis 
## About:
This repository was made for Practical Work 4 of the **INF6422E-Advanced Concepts in Computer Security** course given at Polytechnique Montréal (Winter 2026).

## Objective: 
This lab focuses on semantic, contextual, and trustworthy interpretation of cybersecurity alerts 
using a modern intrusion detection dataset (UNSW-NB15) and a gradient boosting model 
(XGBoost). This lab emphasizes advanced multi-class attack analysis, semantic interpretation of 
alerts, and context-aware security reasoning rather than only improving prediction accuracy. 
Students will analyze how contextual network features (protocol, service, load, timing, and attack 
category) influence intrusion behavior and how machine learning outputs can be interpreted in 
realistic Security Operations Center (SOC) environments.
